# 🌸 International Women's Day 2026 - Beyond the Limits

A professional web development project designed to celebrate and honor the strength, leadership, and innovation of women worldwide. This project was developed as part of my professional tasks.

## 🚀 Live Demo
You can view the interactive experience here:  
👉 [**Live Project Link**](https://hs5042708-web.github.io/international-womens-day/)

---

## ✨ Key Features

* **Interactive Image Slider:** A smooth, JavaScript-powered slider showcasing "The Faces of Change" including Leadership, Innovation, Resilience, and Courage.
* **Inspiration Engine:** An "Ignite Inspiration" feature that uses a custom logic to display empowering quotes for women.
* **Modern Visuals:** A bold Charcoal and Electric Orange theme with neon glow effects and professional typography.
* **Fully Responsive:** Optimized for a seamless experience across all device sizes.

---

## 🛠️ Tech Stack Used

* **HTML5:** Structured with semantic tags for better accessibility and SEO.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and CSS variables for a consistent theme.
* **JavaScript (ES6):** Logic for the image slider and the random quote generation system.

---

## 📂 Project Structure

* `index.html` - The main structure of the web page.
* `style.css` - Custom styling and animations.
* `script.js` - Interactive elements and logic.
